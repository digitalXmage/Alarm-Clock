# Alarm-Clock

# Dependencies

    rtcwake & mpv

# setup
    chmod u+x alarm.sh

# Usage

To save a soundfile for the alarm clock to play

    ./alarm.sh as <path_to_soundfile>
  
To view the saved soundfile
 
    ./alarm.sh ss
  
To set the alarm clock time to wake up to

    ./alarm.sh hh:mm
